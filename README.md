# Parallax Website

A modern and responsive Parallax Scrolling Website built using pure HTML and CSS.

This project demonstrates how to create a smooth parallax scrolling effect using background-attachment: fixed, along with structured content sections and overlay text.
--
## About The Project

This website showcases a parallax scrolling effect where background images move at a different speed than the foreground content while scrolling.

It includes:

Full-screen hero parallax section

Multiple parallax image sections

Light and dark content sections

Centered overlay text

Basic responsive adjustments

This project is focused on practicing layout design, positioning, and CSS background effects.

## Built With

- HTML5

- CSS3
